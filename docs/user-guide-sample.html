Quickstart #

Open this file in any browser: user-guide-sample.html. Use the left nav or scroll; on narrow screens, tap the menu icon to open the drawer.

  • Pipeline — Track deals from sourcing through exit.
  • Screening & analysis — Refine repair estimates and feasibility.
  • Renovation — Manage scope, budgets, and milestones.
💡

Tip

Type “budget”, “pipeline”, or “keyboard” in the search box to see live filtering across all sections and nav links.

Navigation & layout #

The sidebar stays fixed on desktop. IntersectionObserver highlights the active section as you scroll for an app-like “you are here” feel.

Sidebar + content split. Search filters both navigation labels and section visibility. Matched sections fade; non-matches hide completely when the query is specific enough.
Below 900px width, the sidebar becomes a slide-over drawer. Tap outside or a nav link to dismiss. Touch targets meet a minimum 40px for the menu control.
Press / (outside inputs) to jump to search. Escape clears search when focused or closes the mobile drawer.

Built-in search #

Search runs entirely in the browser. Each section declares data-keywords for fuzzy matching alongside visible headings and paragraphs.

Budgets & feasibility #

Underwriting uses ARV, repair reserves, and contingency. Keep investor estimates separate from contractor bids until you move into renovation execution—then reconcile actuals.

📊

Sample metric

70% rule quick check: (ARV × 0.7) − repairs − fees ≈ max offer. Tune percentages per market risk.
